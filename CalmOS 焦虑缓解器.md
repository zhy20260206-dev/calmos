# CalmOS 焦虑缓解器

> 基于 AI 的轻量级焦虑缓解工具 — 3 分钟诊断，即时给出个性化缓解方案
>
> 在线地址：https://zhy20260206-dev.github.io/calmos/

## 一、产品概述

### 一句话介绍

**CalmOS** 是一款基于 AI 的轻量级焦虑缓解工具。用户通过 9 道低负担问卷描述当前焦虑情境，AI 识别焦虑根因，并生成与问题类型相匹配的具体缓解方案，帮助用户从「脑子里很乱」走到「知道下一步做什么」。

### 核心体验

- **先被理解**：AI 解释"你为什么会这样焦虑"，而不是泛泛安慰
- **再被帮助**：输出具体到"现在就做"的动作方案，不是鸡汤式建议
- **形成闭环**：记录焦虑前后变化，让用户看到自己的应对痕迹

### 产品边界

本产品是面向普通焦虑场景的情绪自助工具，不提供医学诊断，不替代心理咨询。检测到高风险内容时，立即中断流程切换为安全引导。

## 三、问卷系统（9 题 × 5 条路径）

### 设计原则

1. **最少题目**：9 题收敛到足够精准推荐，不打扰用户
2. **分支逻辑**：Q1 选择决定 Q2 选项组和 Q4 深挖方向
3. **给情绪名字**：每道题帮助用户把模糊的焦虑感具象化
4. **降低防御**：emoji 圆形图标 + 彩色左边框，替代生硬的选项编号

### 题目架构

| ID | 题目 | 跳过条件 | 选项数 |
|----|------|----------|--------|
| Q1 | 现在让你焦虑的，主要是哪类事？ | 无 | 5 |
| Q2 | 这件事主要是关于？ | Q1=说不清楚 | 5（按 Q1 动态变化） |
| Q3 | 对你最近影响有多大？ | 无 | 4 |
| Q4 | 这件事现在卡在哪里？ | Q1=说不清楚 | 4（按 Q1 动态变化） |
| Q5 | 关于这件事，你目前是？ | 无 | 3 |
| Q6 | 焦虑时身体/情绪状态更接近？ | 无 | 5 |
| Q7 | 思绪主要集中在？ | 无 | 4 |
| Q8 | 现在的你更需要哪种帮助？ | 无 | 4 |
| Q9 | 这件事困扰你多久了？ | 无 | 4 |

### 五条分支路径

| Q1 选择 | Q2 情境示例 | Q4 深挖方向 | 有效题数 |
|---------|-----------|------------|---------|
| 有件事要做 | 工作汇报/考试/生活事务/健康/拖延事项 | 没思路/没动手/卡住了/担心结果 | 9 |
| 在等一个结果或回复 | 面试/考试/体检/他人态度/审批 | 预想最坏/无力/时间感/面对 | 9 |
| 跟某个人或某段关系有关 | 伴侣/父母/好友/同事/前任 | 开口/裂痕/看不懂/让对方不舒服 | 9 |
| 对自己某件事感到后悔或不满 | 后悔说话/错过机会/无法挽回/没进步/比较 | 回放/否定/想修正/担心他人看法 | 9 |
| 说不清楚，就是感觉不太对 | （跳过 Q2、Q4） | （跳过） | 7 |

## 四、模块匹配逻辑

### 核心思路

AI 不自由发挥。我们将 Q1（焦虑类型）和 Q8（需要的帮助）的 **5×4=20 种组合**写死在推荐规则中，AI 严格查表选择 2 个干预模块。这确保了每次推荐的安全性和可预测性。

### 20 种匹配表

| Q1 路径 | 想先平静 | 想理清楚 | 想直接行动 | 想有人陪 |
|---------|---------|---------|-----------|---------|
| 有件事要做 | 呼吸 + 行动计划 | 行动 + 认知重构 | 行动 + 认知重构 | 行动 + 对话脚本 |
| 在等结果 | 呼吸 + 不确定性 | 不确定性 + 认知重构 | 不确定性 + 行动 | 不确定性 + 对话 |
| 人际关系 | 呼吸 + 对话脚本 | 认知重构 + 对话 | 对话 + 认知重构 | 对话 + 价值锚点 |
| 对自己后悔 | 呼吸 + 价值锚点 | 认知重构 + 价值锚点 | 价值锚点 + 行动 | 认知重构 + 对话 |
| 说不清楚 | 呼吸 + 价值锚点 | 价值锚点 + 认知重构 | 价值锚点 + 行动 | 价值锚点 + 对话 |

## 五、六个干预模块

所有模块基于认知行为疗法（CBT）和接纳承诺疗法（ACT）的循证技术设计。每次推荐 2 个模块以「步骤 1/2 → 步骤 2/2」串联引导，完成一个自动进入下一个。

### 模块 1：行动计划（action_plan）

| 维度 | 内容 |
|------|------|
| **解决什么** | 任务模糊 → 拖延 → 焦虑积累 |
| **理论来源** | CBT 行为激活 |
| **给用户什么** | 把大任务拆成 3-4 个"5 分钟就能完成"的小步骤 |
| **用户怎么做** | 每完成一步点击打勾 ✅，全部勾完激活完成按钮 |
| **耗时** | 约 5-10 分钟 |

### 模块 2：呼吸稳定（breathing）

| 维度 | 内容 |
|------|------|
| **解决什么** | 焦虑急性生理反应：心跳加速、呼吸急促、胸闷 |
| **理论来源** | 生理调节（副交感神经激活） |
| **给用户什么** | 4-7-8 呼吸节奏（吸气 4 秒 → 屏住 7 秒 → 呼气 8 秒） |
| **用户怎么做** | 跟着屏幕上的圆形 blob 动画吸气/呼气，3 个循环自动完成 |
| **耗时** | 约 1 分钟 |

### 模块 3：不确定性管理（uncertainty_management）

| 维度 | 内容 |
|------|------|
| **解决什么** | 等待结果时反复预想最坏情况，脑子停不下来 |
| **理论来源** | 不确定性耐受训练 + 暴露与反应预防 |
| **给用户什么** | 三场景分析（最好/中等/最坏情况各一个应对）+ 等待期可做的小事 + 信息接触限制建议 |
| **用户怎么做** | 横滑浏览三张场景卡片，阅读行动建议 |
| **耗时** | 约 3-5 分钟 |

### 模块 4：对话脚本（conversation_script）

| 维度 | 内容 |
|------|------|
| **解决什么** | 人际焦虑：有话想说但不知道怎么开口 |
| **理论来源** | 人际效能训练 |
| **给用户什么** | 温和版 + 直接版两种开场话术 + 沟通渠道和时机建议 + 48 小时等待窗口建议 |
| **用户怎么做** | 阅读两种话术，获得具体可用的开口方式 |
| **耗时** | 约 3-5 分钟 |

### 模块 5：认知重构（cognitive_reframe）

| 维度 | 内容 |
|------|------|
| **解决什么** | 自我否定的自动化负面想法："我是不是很差劲" |
| **理论来源** | CBT 三栏表法 |
| **给用户什么** | 事实 vs 大脑加戏的并排对比 → 识别思维扭曲模式 → 更接近真相的替代想法 → 好友视角："如果是好朋友经历这件事，你会对 ta 说什么" |
| **用户怎么做** | 阅读对比卡片（绿色事实 / 红色加戏），可复制"对好朋友说的话" |
| **耗时** | 约 3-5 分钟 |

### 模块 6：价值锚点（value_anchor）

| 维度 | 内容 |
|------|------|
| **解决什么** | 存在迷茫：说不清的焦虑、空心感、失控感 |
| **理论来源** | ACT 接纳承诺疗法 |
| **给用户什么** | 帮情绪取名字 + 一个今天就能做的小行动 + 5-4-3 感官接地练习 + 一句许可语句 |
| **用户怎么做** | 看见大字号情绪名称 → 读取锚点行动 → 跟着做感官接地 |
| **耗时** | 约 3-5 分钟 |

## 六、结果与反馈

### 结果页

AI 分析完成后展示：

- **根因标签**：10 字以内焦虑根因提炼
- **解读摘要**：80-120 字个性化解读，像朋友复盘，不是模板套话
- **证据标签**：3 个来自用户答案的关键词标签
- **2 张模块推荐卡**：每张含标题、推荐理由、预计时长，标识为第 1 步/第 2 步
- **收尾鼓励语**：20-35 字，温暖有力但不油腻

### 反馈页

- **焦虑自评**：0-10 滑条，前后分数对比动画
- **帮助程度**：3 选 1（有帮助/一般/没什么帮助）
- **智能反馈语**：分数下降时"有一点变化，就已经很重要"，分数未变时"没关系，找到不适合的方法也是进展"
- 保存后返回主页，历史记录即时刷新

## 七、设计语言

### 视觉理念：手绘涂鸦 × 马卡龙

**告别"白大褂"感**。不选择严肃的医学/科技风格，而是用糖果色 + 微旋转卡片 + 有机 blob 形状传递温暖和安全。看起来更像"朋友聊天"而非"看医生"。

### 色彩系统

| 颜色 | 色值 | 感受 | 用途 |
|------|------|------|------|
| 奶油黄 | `#FFEAA7` | 温暖、希望 | 用户头像 blob、高亮标签 |
| 薄荷绿 | `#A8E6CF` | 平静、恢复 | 按钮阴影、进度条、选中态 |
| 深绿 | `#6ECBA4` | 成长、完成 | 步骤完成标记 |
| 天蓝 | `#A8D8EA` | 信任、清晰 | 信息卡片、备选模块 |
| 珊瑚 | `#FFB4A2` | 能量、跳色 | 第 1 步标识、重要提示 |
| 淡紫 | `#C5B4E3` | 温柔、接纳 | 认知重构区域点缀 |
| 暖白 | `#FFFDF9` | 干净、安静 | 主背景 |

### 交互原则

1. **一次只做一件事**：每个屏幕只有一个核心操作
2. **不给太多选择**：每题 4-5 个选项，符合 Hick's Law
3. **每步有进度条**：问卷进度条 + 干预模块步骤进度条
4. **所有操作可撤销**：返回按钮无处不在
5. **文案像朋友说话**：用"你可能在想"而非"你的认知扭曲是"
6. **焦虑不叠加**：精美的过渡动画让 AI 等待时间变得可接受

## 八、安全与伦理

### 危机响应

当检测到高风险信号时，不展示任何 AI 分析，直接弹出危机引导页：
- 全国心理援助热线：400-161-9995
- 北京心理危机研究与干预中心：010-82951332
- 引导文案：「你不需要一个人扛着。拨出去只需要一秒钟。」

### 伦理边界

- **不是医疗产品**：明确在产品中声明不替代专业心理咨询或医疗诊断
- **不收集隐私**：不收集姓名、手机号、身份证等可识别个人身份的信息
- **用户匿名**：通过昵称和 emoji 头像区分用户，无需真实身份
- **AI 透明**：用户知晓分析结果来自 AI，产品名 CalmOS 暗示工具/操作系统定位而非医生角色
- **数据自主**：数据默认保存在用户本地设备，云端同步为可选项
