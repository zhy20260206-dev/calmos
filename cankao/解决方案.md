以下是基于 CBT 循证研究与现有焦虑缓解产品的梳理，按焦虑类型分类，每种都给出具体干预方案。

***

## 一、任务堆积型（拖延焦虑）

**核心机制**：任务越大越模糊 → 越难开始 → 焦虑积累 → 更难开始（恶性循环）

### 干预方案

**① 行为激活（Behavioral Activation）** [onebright](https://onebright.com/advice-hub/news/cbt-overcoming-procrastination/)
把"做完这件事"拆成不超过 3 步，每步限时 ≤20 分钟，只关注"开始"而不是"完成"。
- 核心操作：列出最小可行第一步 → 设定具体时间段 → 完成后记录成就感
- 对应工具：Todoist、Notion 任务拆解

**② 5 分钟规则（CBT Procrastination）** [cogbtherapy](https://cogbtherapy.com/procrastination-los-angeles)
告诉自己"只做 5 分钟"，利用启动惯性打破回避。
- 核心操作：设定 5 分钟计时器 → 只承诺开始，不承诺完成
- 产品形态：倒计时页面 + 完成确认

**③ 刺激控制（Stimulus Control）** [onebright](https://onebright.com/advice-hub/news/cbt-overcoming-procrastination/)
减少让人分心的环境刺激（手机、通知）来降低回避行为的触发频率。
- 核心操作：开始前提示关闭通知 / 手机放远 + 只开一个窗口

**④ 番茄工作法变体** [centerforcbtinnyc](https://centerforcbtinnyc.com/cbt-for-procrastination-overcoming-self-sabotage-and-boosting-productivity/)
25 分钟专注 + 5 分钟休息，以"时间段"而非"任务完成"为单位考量。

## 二、等待失控型（不确定性焦虑）

**核心机制**：结果不可控 → 反复预演最坏情况 → 注意力被占用 → 更多预演

### 干预方案

**① 不确定性耐受训练（Uncertainty Tolerance）** [pmc.ncbi.nlm.nih](https://pmc.ncbi.nlm.nih.gov/articles/PMC8475916/)
通过列出所有可能结果（好/中/坏），把"未知"变成"有范围的已知"，降低模糊感带来的恐惧。
- 核心操作：写出 3 种结果 → 每种结果写出第一步应对 → 感知到"无论如何我都能应对"

**② 注意力重新分配（Attentional Refocusing）** [pmc.ncbi.nlm.nih](https://pmc.ncbi.nlm.nih.gov/articles/PMC8475916/)
在等待期间，主动安排与等待结果无关的活动填充时间，减少反刍。
- 核心操作：列出 2 件等待期间可以继续推进的事 → 承诺每天做其中一件

**③ 信息接触限制** [mayoclinic](https://www.mayoclinic.org/zh-hans/diseases-conditions/generalized-anxiety-disorder/diagnosis-treatment/drc-20361045)
设定每天只查看一次相关信息的规则，打破反复刷新的强迫行为。
- 产品形态：提示用户设置"查看时间窗口"（如每天下午 5 点看一次）

**④ 最坏情景预演（Worst-case Scenario Planning）**
主动想清楚最坏的结果发生后"我能怎么做"，从而降低对它的恐惧感。 [nationalsocialanxietycenter](https://nationalsocialanxietycenter.com/cognitive-behavioral-therapy/social-anxiety-strategies/)

## 三、人际担忧型（关系焦虑）

**核心机制**：关系中出现不确定信号 → 反复解读对方行为 → 不知道怎么回应 → 焦虑加剧

### 干预方案

**① 非防御性沟通脚本（IPT 人际疗法）** [emotiv](https://www.emotiv.com/zh/neuroscience/anxiety-therapy)
帮助用户准备"如何开口"，使用第一人称情感表达（"我感到…"而不是"你让我…"），降低对话门槛。 [nationalsocialanxietycenter](https://nationalsocialanxietycenter.com/cognitive-behavioral-therapy/social-anxiety-strategies/)
- 核心操作：明确这次对话目的 → 提供 2 种风格话术（温和版 / 直接版）

**② 行为实验（Behavioral Experiments）** [dallascbt](https://dallascbt.com/2023/10/5-effective-cbt-based-exercises-for-managing-anxiety/)
鼓励用户实际去做一件小事（发一条消息）来检验自己的恐惧是否真实，而不只是在脑子里预演。

**③ 去灾难化（Decatastrophizing）** [acp-mn](https://acp-mn.com/about-acp/blog/cbt-exercises-anxiety/)
引导用户识别"他一定觉得我很烦"这类灾难化推断，检验证据，生成更现实的解读。

**④ 等待窗口设定**
关系类焦虑容易陷入反复发消息 / 反复查看的循环。提示用户设定"48 小时等待期"，然后再判断是否需要跟进。

***

## 四、自我否定型（反刍焦虑）

**核心机制**：发生一件事 → 对事件做灾难化解读 → 泛化到自我评价（"我就是这种人"）→ 情绪持续低落

### 干预方案

**① 认知重构（Cognitive Restructuring）** [acp-mn](https://acp-mn.com/about-acp/blog/cbt-exercises-anxiety/)
把"我很失败"这类整体性否定拆解为：
- 事件层（发生了什么）
- 解读层（我怎么解读它）
- 现实层（更平衡的说法是什么）

> 工具：三栏表——自动负面想法 / 认知扭曲类型 / 更现实的替代想法 [acp-mn](https://acp-mn.com/about-acp/blog/cbt-exercises-anxiety/)

**② 自我宽容视角转换（Self-Compassion）** [emotiv](https://www.emotiv.com/zh/neuroscience/anxiety-therapy)
用"如果是你的好朋友经历了这件事，你会怎么对他说？"的视角重写对自己的评价，打破双重标准。

**③ 去中心化练习（Defusion）** [emotiv](https://www.emotiv.com/zh/neuroscience/anxiety-therapy)
让用户把想法从"我就是个失败者"变成"我注意到自己有一个想法，说我是个失败者"，制造观察者距离。

**④ 小补偿行动**
如果问题可以部分修复（道歉、补做某事），帮助用户制定一个最小补救步骤，把反刍能量转化为行动。

***

## 五、存在迷茫型（空心感 / 意义缺失焦虑）

**核心机制**：无法识别焦虑来源 → 感到失控和空洞 → 回避思考 → 麻木积累

### 干预方案

**① 情绪命名（Affect Labeling）**
研究表明，仅仅给情绪贴上准确的标签，就能降低杏仁核的激活程度（"命名它，驯服它"）。 [pmc.ncbi.nlm.nih](https://pmc.ncbi.nlm.nih.gov/articles/PMC8475916/)
- 核心操作：提供 8 种情绪状态描述，让用户选"最接近的那种"

**② 价值小锚点（ACT 接纳承诺疗法）** [emotiv](https://www.emotiv.com/zh/neuroscience/anxiety-therapy)
不试图解决"人生方向"这个大问题，而是在 4 个价值维度（健康 / 关系 / 成就 / 兴趣）中，找一件今天可以做的极小事。

**③ 活动安排（Activity Scheduling）** [acp-mn](https://acp-mn.com/about-acp/blog/cbt-exercises-anxiety/)
迷茫状态下，从日程入手比从思考入手更有效。安排一件具体的、有可能带来愉悦或成就感的活动。

**④ 正念落地技术（Grounding）** [dallascbt](https://dallascbt.com/2023/10/5-effective-cbt-based-exercises-for-managing-anxiety/)
5-4-3-2-1 感官落地法：列出 5 个看到的 / 4 个摸到的 / 3 个听到的，把注意力拉回当下，打断空转。

## 六、现有产品的策略对比

| 产品 | 核心策略 | 不足之处 |
|---|---|---|
| Calm  [calm](https://www.calm.com) | 呼吸练习 + 冥想 + 睡眠故事 | 所有人同一方案，不区分焦虑类型 |
| Headspace  [headspace](https://www.headspace.com) | 冥想课程 + 专注引导 | 偏长期习惯，缺乏即时干预 |
| Woebot  [mindspacebd](https://www.mindspacebd.com/mental-health-o-pedia/best-apps-for-managing-anxiety-and-depression) | AI 对话 + CBT 技术 | 聊天为主，输出不够具体可执行 |
| MindShift  [sacd.sdsu](https://sacd.sdsu.edu/cps/self-care/mental-health-apps) | CBT 工具箱（多种工具） | 工具多但缺乏个性化推荐逻辑 |
| Sanvello  [sacd.sdsu](https://sacd.sdsu.edu/cps/self-care/mental-health-apps) | 日记 + 情绪追踪 + CBT | 需要持续使用，即时效果弱 |

**CalmOS 的差异化空间**：现有产品要么"同一方案给所有人"，要么"工具多但不知道用哪个"。CalmOS 的价值在于**先识别类型，再给对应工具**，且每次只给一个具体动作，不让用户选择困难。
