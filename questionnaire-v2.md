## 各题选项详情

### Q1 · 焦虑类型（5 选项 · 静态 · 全路径）

1. 有件事要做，但还没开始
2. 在等一个结果或回复
3. 跟某个人或某段关系有关
4. 对自己某件事感到后悔或不满
5. 说不清楚，就是感觉不太对

### Q2 · 具体情境（8 选项 · 按 Q1 动态）

#### 若 Q1 = 有件事要做，但还没开始

1. 季度汇报 / 年终总结
2. 重要考试 / 证书备考
3. 搬家 / 装修 / 大扫除
4. 看病 / 体检 / 预约
5. 记账 / 报税 / 行政手续
6. 健身 / 饮食 / 作息调整
7. 回复重要消息或邮件
8. 创意输出 / 写作 / 设计

#### 若 Q1 = 在等一个结果或回复

1. 工作面试结果
2. 学校录取通知
3. 考试 / 竞赛成绩
4. 体检 / 病理报告
5. 客户 / 合作方回复
6. 贷款 / 签证 / 审批
7. 告白 / 道歉后等回应
8. 法院 / 仲裁结果

#### 若 Q1 = 跟某个人或某段关系有关

1. 伴侣或暧昧对象
2. 父母或长辈
3. 子女或晚辈
4. 闺蜜 / 兄弟 / 好友
5. 领导 / 导师 / 上级
6. 同事 / 合伙人 / 同学
7. 邻居 / 室友 / 日常接触
8. 前任 / 断联的人

#### 若 Q1 = 对自己某件事感到后悔或不满

1. 一时冲动说了伤人的话
2. 错过了一个重要机会
3. 觉得自己不够好 / 不够努力
4. 做了一件无法挽回的决定
5. 浪费了很多时间
6. 对自己外貌 / 身体不满意
7. 总觉得不如别人
8. 对未来方向后悔

#### 若 Q1 = 说不清楚（❌ 本题目被跳过）

### Q3 · 影响程度（4 选项 · 静态 · 全路径）

1. 有一点影响，但还能正常做事
2. 有些影响，有时会分神
3. 影响比较大，经常难以集中注意力
4. 影响非常大，几乎做不了其他事

### Q4 · 卡在阶段（4 选项 · 静态 · Q1=说不清楚时跳过）

1. 完全没思路，不知道怎么做
2. 有思路，但一直没动手
3. 动手了，但中途卡住推不下去
4. 快做完了，但对结果很不放心

### Q5 · 社交支持（3 选项 · 静态 · 全路径）

1. 没有跟任何人说过
2. 说了，但没有得到真正帮助
3. 身边有人支持，但问题本身还在

### Q6 · 身体信号（5 选项 · 静态 · 全路径）

1. 心跳加速或呼吸变化
2. 头紧、头痛、注意力涣散
3. 胃部不适或没食欲
4. 肩颈紧绷或肌肉僵硬
5. 身体没太大变化，主要是脑子停不下来

### Q7 · 思维方向（4 选项 · 静态 · 全路径）

1. 过去：反复回放已经发生的事
2. 现在：眼前一堆事乱成一团
3. 未来：一直预想还没发生的事
4. 过去/现在/未来之间来回跳

### Q8 · 需要帮助（4 选项 · 静态 · 全路径）

1. 先让情绪和身体平静下来
2. 理清楚这件事，弄清下一步怎么办
3. 直接开始行动，打破卡住状态
4. 有人陪着，不想一个人扛

### Q9 · 持续时间（4 选项 · 静态 · 全路径）

1. 今天才开始
2. 最近几天
3. 已经好几周了
4. 很长一段时间了

### Q10 · 情绪状态（8 选项 · 静态 · 全路径）

1. 越想越烦，脑子停不下来
2. 浑身没劲，什么都不想干
3. 想逃避，躲起来就好
4. 心慌意乱，身体不对劲
5. 说不清的难受，堵得慌
6. 麻木了，感觉不到什么
7. 有点生气或不甘心
8. 就是觉得很孤单
